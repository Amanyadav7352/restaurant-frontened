# 🍽️ Responsive Restaurant Website

A modern and  responsive restaurant website built using **HTML and CSS**.
This project focuses on clean UI design, responsive layouts, and real-world web development practices.

---

## 🚀 Live Demo
👉 https://html-css-restaurant.netlify.app/

---

## 📌 Features

* ✅ Responsive design (Mobile, Tablet, Desktop)
* ✅ Modern UI layout using Flexbox & Grid
* ✅ Hero section with call-to-action
* ✅ Food menu grid layout
* ✅ Feature highlights (Discount, Fresh Food, Delivery)
* ✅ Sticky navigation bar
* ✅ Footer with links and social icons

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox + Grid)
* Google Fonts

---

## 📱 Responsiveness

This website is optimized for:

* 📱 Mobile devices (≤ 400px)
* 📲 Tablets (≤ 768px)
* 💻 Desktop screens
 # Further improvements can be made to enhance responsiveness,
  such as implementing a mobile-first approach, adding a dynamic navigation menu,
  and optimizing layouts for more screen sizes.
